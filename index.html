译函 (Translate Mail) — 隐私政策

生效日期：2026 年 6 月 6 日

我们不收集任何数据

译函是一款完全本地运行的 macOS 邮件翻译工具。我们不收集、不存储、不传输任何用户个人信息或邮件内容。

邮件内容

译函通过 macOS 自动化接口读取你在 Apple Mail 中选中的邮件内容，仅用于翻译和分析。邮件内容不会被发送到我们的服务器，因为我们没有服务器。

AI 翻译服务

当使用 AI 大模型翻译时，邮件内容会被发送到你自行配置的第三方 AI 服务商（如 Moonshot、DeepSeek、Xiaomi MiMo 等）。这些服务商的数据处理受其各自隐私政策约束。译函不经手、不中转这些数据。

系统翻译

当使用 macOS 系统翻译时，翻译完全在本地离线完成，不涉及任何网络传输。

API Key 存储

你配置的 AI 服务商 API Key 以 AES-GCM 加密方式存储在本地（~/Library/Application Support/译函/），不会上传到任何地方。

Translate Mail — Privacy Policy

Effective Date: June 6, 2026

We Do Not Collect Any Data

Translate Mail is a fully local macOS email translation tool. We do not collect, store, or transmit any personal information or email content.

Email Content

Translate Mail reads the selected email in Apple Mail through macOS automation APIs, solely for translation and analysis. Email content is never sent to our servers — we have no servers.

AI Translation Services

When using AI translation, email content is sent to third-party AI providers (such as Moonshot, DeepSeek, Xiaomi MiMo, etc.) that you configure yourself. Data handling by these providers is governed by their respective privacy policies. Translate Mail does not relay or store this data.

System Translation

When using macOS system translation, all processing is done locally and offline, with no network transmission involved.

API Key Storage

API keys you configure are stored locally with AES-GCM encryption (~/Library/Application Support/译函/) and are never uploaded anywhere.
